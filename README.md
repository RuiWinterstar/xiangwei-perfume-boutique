# Xiangwei Perfume Boutique

Xiangwei is a luxury perfume boutique website developed using HTML, CSS, and JavaScript. This project was created as a university final project and demonstrates front-end web development, responsive layouts, and client-side e-commerce features.

## Live Demo

🌐 https://xiangwei.my.id
🌐 https://ruiwinterstar.github.io/xiangwei-perfume-boutique/

## Features

- Luxury landing page
- Product catalog
- Product detail pages
- Shopping cart (Local Storage)
- Checkout system
- Order history
- Portfolio gallery
- Image lightbox viewer
- Google Sheets comment integration
- Responsive design

## Built With

- HTML5
- CSS3
- JavaScript
- Google Apps Script
- Google Sheets
